# PEGASUS Typewriter

A minimalist writing plugin for PEGASUS.

## Features

- Clean writing interface
- Autosave system
- Page navigation
- Dialogue formatting
- Title formatting
- PDF export
- DOCX export

## Requirements

- customtkinter
- fpdf2
- python-docx

## Entry Point

def run(nexus):
    app = PegasusTypewriter(nexus)
